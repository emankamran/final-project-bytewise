# final-project-bytewise

I implemented frontend state management using Redux, enabling efficient data handling and delivering a seamless user experience. The application incorporates email sending functionality, complete authentication flow (registration, login, forget password with email, and reset password) with robust security measures and user-friendly features. JWT tokens are used for authentication, and web sockets are integrated for real-time communication. Additionally, I implemented cron jobs for automated tasks, such as newsletter email sending, and utilized email queuing to optimize backend processes and ensure smooth email delivery.

## Prerequisites
- Redis Server

## Getting Started

### Server

1. Navigate to the server directory
2. Install server dependencies
3. Start the server

   
### Client

1. Navigate to the client directory
2. Install client dependencies
3. Start the client

   
### Redis Server

1. Ensure Redis Server is installed on your local machine.
2. Start the Redis server: redis-server

## Usage

Open your browser and navigate to `http://localhost:3000` to access the application.
